技术栈

人间浊物 琅上清书 是一个基于纯静态文件构建的个人博客：

  • 纯 HTML + CSS + JavaScript，无框架依赖
  • 数据以 JSON 格式存储，便于维护与迁移
  • 支持 Markdown 格式的文章编辑
  • 内置深色/浅色双主题
  • 集成 Giscus 评论系统（基于 GitHub Discussions,有待完善~）
  • 图形化编辑后台，支持图片上传与数据导出

博客源码托管于 GitHub， 通过 GitHub Pages 部署发布。
